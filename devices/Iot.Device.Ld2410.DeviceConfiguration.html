    Declaration

    Property Value
    Type Description
    byte
    Remarks

    Gate numbers start from 0. So a radar reporting MaxDistanceGateIndex of 8 means it has 9 gates. Each gate covers a distance of 75cm. If a module reports max distance gate of 8, then it covers 9x75=675cm or 6.75m.
