    Enum BaudRate

    Defines the set of Baud Rates that the device's serial port supports.

    Namespace: Iot.Device.Ld2410
    Assembly: Iot.Device.Ld2410.dll
    Syntax
    public enum BaudRate : ushort
    Remarks

    The serial communication of LD2410 uses little-endian format. All the values in this enum are in big-endian to match the manufacturer docs. Use WriteUInt16LittleEndian(SpanByte, ushort) before writing to the serial port to ensure that the byte representation will be converted to little-endian if the code is running on a big-endian CPU.
