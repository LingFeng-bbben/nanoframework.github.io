    Class SparkFunLcd

    LCD library for SparkFun RGB Serial Open LCD display (sizes 20x4 or 16x2) with I2C connection for product information see https://www.sparkfun.com/products/16398 code based on https://github.com/sparkfun/OpenLCD.
