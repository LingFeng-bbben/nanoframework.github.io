    Namespace nanoFramework.Logging.Syslog

    Classes

    SyslogClient

    Syslog Client that sends logs to a Syslog server following RFC3164 format.

    SyslogLogger

    ILogger implementation that send message log to a Syslog server following RFC3164 convention. While not mandatory it is recommended to use SyslogLoggerFactory to create the SyslogLogger instances.

    SyslogLoggerFactory

    ILoggerFactory implementation that creates SyslogLogger instances. The logger factory initializes an underlying SyslogClient that will be shared by all SyslogLogger instances it will create.

    Enums

    Facility

    Available facility types (RFC 3164).

    Severity

    Severity represents the Severity in the Syslog messages. This corresponds to the values defined in RFC 3164 except None which means that the message won't be sent.
